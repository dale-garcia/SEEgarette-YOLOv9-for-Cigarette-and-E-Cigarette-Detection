# SEEgarette-YOLOv9-for-Cigarette-and-E-Cigarette-Detection 

SEEgarette: YOLOv9 for Cigarette and E-Cigarette Detection is a computer vision–based detection system developed for identifying cigarettes and E-Cigarettes in real-time environments using deep learning object detection models. This research explores the effectiveness of YOLOv9 compared with YOLOv8 in detecting smoking-related objects within public and crowded spaces. This study delved into YOLOv9's object detection capability, which can enhance public health monitoring and safety enforcement efforts. Improved detection accuracy supports more efficient identification that could amplify regulation of harmful substances in public spaces. 

A custom dataset consisting of 1,500 images was utilized in the study, with 400 images sourced online and 1,100 images collected independently. To support data collection and sampling, a field survey involving 20 vape shops along Quezon and Coloocan City was conducted under the assumption that smoking-related activity and object visibility are more prevalent in areas surrounding vape-oriented commercial establishments. This approach contributed to targeted image acquisition and contextual dataset diversity for model training and validation. The YOLOv9 model was trained using a batch size of 16 over 25 epochs to evaluate real-time cigarette and E-Cigarette detection performance.

This deep learning–based object detection study for automated cigarette and E-Cigarette was presented at ICITE 2024 (International Conference on Information Technology Education) organized by the Philippine Society of Information Technology Educators (PSITE) held in Solbridge International School of Business in South Korea.
